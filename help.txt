=============================================
Программа анализа рентгеновских линз: Руководство
=============================================

1. Общее описание
-----------------
Программа предназначена для анализа и моделирования оптических свойств 
мультипризматических рентгеновских линз на основе данных профилометрии.

Основные функции:
- Визуализация профиля линзы
- Расчет функции толщины материала
- Анализ амплитуды и фазы прошедшего излучения
- Моделирование распространения волны на различных расстояниях

2. Требования к входным данным
------------------------------
Программа работает с данными профилометрии в формате CSV. Файл должен содержать:

Формат данных:
- Первый столбец: координата X (мм)
- Второй столбец: высота профиля Y (мкм)

Требования к измерениям:
1. Разрешение: не менее 100 точек на период структуры
2. Диапазон: должен охватывать минимум 5 полных периодов линзы
3. Шум: RMS шума не должен превышать 0.5% от высоты профиля
4. Калибровка: данные должны быть корректно откалиброваны

Пример корректных данных:
0.000, 12.345
0.001, 12.350
...
10.000, 15.678

3. Работа с программой
----------------------

3.1. Вкладка "Профиль линзы"
---------------------------
- Отображает исходные данные профилометрии
- Позволяет сравнить профиль до и после поворота
- Параметры:
  * Угол раскрытия: угол поворота профиля (0.1-5°)

3.2. Вкладка "Функция толщины"
------------------------------
- Показывает расчетную толщину материала вдоль линзы
- Режимы отображения:
  1. Реальный профиль - по данным измерений
  2. Теоретический профиль - идеализированная парабола
  3. Сравнение - наложение обоих профилей

3.3. Вкладка "Интенсивность"
----------------------------
- Анализ волны непосредственно после линзы
- Управление:
  * Выбор профиля (реальный/теоретический/оба)
  * Выбор отображаемых параметров:
    - Амплитуда
    - Фаза
    - Оба параметра вместе

3.4. Вкладка "Волна на расстоянии"
----------------------------------
- Моделирование распространения волны
- Параметры:
  * Расстояние: регулируется слайдером (0.5F-2F)
  * Отображение: амплитуда/фаза/оба

3.5. Вкладка "Параметры"
------------------------
- Сводная таблица расчетных параметров:
  * Геометрические характеристики линзы
  * Оптические свойства материала
  * Фокусные характеристики

4. Интерпретация данных профилометрии
------------------------------------
4.1. Ключевые параметры профиля:
- Высота зуба (y_t): максимальная высота структуры
- Ширина периода: расстояние между соседними вершинами
- Асимметрия: разница между левым и правым склонами
- Шероховатость: среднеквадратичное отклонение от идеального профиля

4.2. Типичные артефакты:
1. Сдвиг нулевого уровня - требует коррекции offset
2. Термодрейф - проявляется как линейный наклон базы
3. Вибрации - высокочастотные колебания профиля
4. Загрязнения - резкие выбросы в данных

5. Рекомендации по анализу
--------------------------
1. Всегда проверяйте исходный профиль на артефакты
2. Сравнивайте реальные данные с теоретической моделью
3. Анализируйте как амплитуду, так и фазу волны
4. Исследуйте изменение волнового фронта на разных расстояниях
5. Особое внимание уделяйте областям:
   - Вершины зубьев (влияют на фокусировку)
   - Основания профиля (определяют общие потери)

6. Сохранение результатов
-------------------------
Программа позволяет сохранить:
- Все графики в формате PNG
- Расчетные параметры в текстовом файле
- Данные для последующей обработки

Для корректной работы рекомендуется:
1. Сохранять графики перед изменением параметров
2. Архивировать полный набор результатов для каждого образца
3. Включать в имя файла основные параметры (энергия, материал)

7. Техническая поддержка
------------------------
При возникновении проблем:
1. Проверьте формат входных данных
2. Убедитесь в корректности всех параметров
3. Для сложных случаев предоставьте:
   - Исходный файл данных
   - Скриншот проблемы
   - Значения всех параметров
